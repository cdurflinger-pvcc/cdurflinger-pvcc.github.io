Operating Systems Terminology

Terms Definiiton
Operating System the software tha controls the overall operation of a computer
Job queue jobs residing in mass storage waiting for execution
FIFO First-in,first-out
Interactive processing allows a program being executed to carry on a dialogue with the user through remote terminals
Time-sharing operating systems that provide service to multiple users at the same time
Multiprogramming time is divided into intervals and each job is restricted to only one interval at a time
Application software a computer software package that performs a specific function directly for an end user
System software performs tasks that are common in computer systems in general
Utility software programs performing activities that are fundamental to computer installations
Graphical user interface (GUI) objects to be manipulated are represented poctorially on the displays as icons.
Shell communicated with users through textual messages using a keyboard and monitor screen
Kernal internal part of an operating system
Device manager allows users to remotely manage the hardware attached to a computer
File manager coordinate the use of the machine's mass storage facilities
Memory manager charged with the task of coordinating the machine's use of main memory
Virtual memory large "fictional" memory space created by paging
Boot strapping performed by a computer each time it is turned on
ROM small portion of the computer's main memory where the CPU expects to find its intial program
Deadlock condition in which two or more processes are blocked from progressing
Spooling the technique of holding data for output at a later but more convenient time
Super user allows the user to remove processes
Multi-factor authentication authentication method that requires a user to provide two or more verfication factors
